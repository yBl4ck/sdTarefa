Para Executar a Aplicaçao:

-Abra o PowerShell do VS
-Execute o Comando:
>> .\mvnw.cmd clean package (Assim voce Asegura limpar tudo e qualquer otro arquivo compilado Anteriormente e executa as provas Unitarias)

-Execute o Programa :
>> .\mvnw.cmd spring-boot:run (Com isso aqui voçe vai Iniciar o Servidor Spring Boot)

-Abra o seu Navegador:
Digite http://localhost:8081 (Listo seu App estara rodando 100% funcional)

-Requisitos:

. Java 17 ou superior instalado e configurado (JAVA_HOME definido).
. Maven Wrapper (mvnw.cmd) incluso no projeto (não é necessário instalar Maven manualmente).
. Porta 8081 livre no sistema.
